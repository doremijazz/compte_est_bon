# Le compte est bon

Jeu en console le compte est bon : **Le compte est bon** est un jeu de calcul 
mental qui consiste à atteindre un nombre cible à partir de six nombres 
aléatoires et des quatre opérations (addition, soustraction, multiplication,
division).

Le programme gènére une valeur cible entre **101 et 999**, ainsi que 6 plaques dans
une listes de plaques contenant chaque nombre de 1 à 10, 2 fois, et les nombres 25, 
50, 75 et 100 en un seul exemplaire. 

## Fonctionnalités

- Génération aléatoire d’une cible entre 101 et 999
- Tirage de six plaques
- Opération mathématique : Addition, Soustraction, Multiplication, Division
- Remplacement des deux plaques utilisées par le résultat obtenu
- Détection automatique de la victoire
- Calcul de l’écart avec la cible lorsque toutes les plaques ont été utilisées

## Structure du projet
```text
compte_est_bon/
├── main.py

```

## Fin de partie

La partie se termine dans deux situations :

- le résultat obtenu est exactement égal à la cible ;
- il ne reste plus qu’une seule plaque.
